# Portfolio Design Style Guide

## Design Philosophy

**Theme**: Cyberpunk Gaming Aesthetic with Modern Web Development Focus
**Target Audience**: Tech recruiters, potential clients, gaming community, fellow developers
**Emotional Impact**: Professional yet exciting, showcasing technical skills through immersive design

## Color Palette

**Primary Colors**:
- Deep Space Blue: #0a0e27 (main background)
- Electric Cyan: #00d4ff (primary accent, links, highlights)
- Neon Purple: #8b5cf6 (secondary accent, buttons)
- Bright White: #ffffff (text, contrast elements)

**Secondary Colors**:
- Dark Gray: #1a1a2e (cards, sections)
- Medium Gray: #16213e (borders, subtle elements)
- Success Green: #10b981 (success states, positive indicators)
- Warning Orange: #f59e0b (attention elements)

## Typography

**Display Font**: "Orbitron" - Futuristic, tech-inspired for headings
**Body Font**: "Inter" - Clean, readable for content text
**Code Font**: "JetBrains Mono" - For code snippets and technical content

**Hierarchy**:
- H1: 3.5rem, bold, Orbitron
- H2: 2.5rem, semibold, Orbitron  
- H3: 1.8rem, medium, Orbitron
- Body: 1rem, regular, Inter
- Small: 0.875rem, regular, Inter

## Visual Language

**Gaming Elements**:
- Neon glow effects on interactive elements
- RGB lighting-inspired gradients
- Circuit board patterns as decorative elements
- Gaming controller icons and UI elements
- Retro-futuristic design patterns

**Tech Integration**:
- Code syntax highlighting
- Terminal-style command lines
- GitHub-style contribution graphs
- Tech stack logos and badges
- Loading animations with progress bars

## Visual Effects

**Used Libraries**:
1. **Anime.js** - Smooth animations and transitions
2. **PIXI.js** - Particle effects and interactive backgrounds
3. **ECharts.js** - Skill visualization charts
4. **Typed.js** - Typewriter text effects
5. **Splitting.js** - Text animation effects
6. **Splide.js** - Image carousels and sliders
7. **p5.js** - Creative coding backgrounds

**Animation Effects**:
- Typewriter animation for hero text
- Neon glow pulse effects on buttons
- Particle system background
- Skill bars animation on scroll
- Project cards with 3D tilt hover
- Smooth page transitions
- Loading screen animations

**Header Background Effect**:
- Animated particle field using p5.js
- Floating geometric shapes with neon outlines
- Subtle parallax movement on mouse interaction
- Color cycling between cyan and purple

## Layout Structure

**Grid System**: CSS Grid with 12-column layout
**Breakpoints**: 
- Mobile: 320px - 768px
- Tablet: 768px - 1024px  
- Desktop: 1024px+

**Spacing Scale**:
- xs: 0.25rem
- sm: 0.5rem
- md: 1rem
- lg: 1.5rem
- xl: 2rem
- 2xl: 3rem

## Component Styling

**Cards**: Dark background with neon borders, hover lift effect
**Buttons**: Gradient backgrounds with glow effects
**Forms**: Dark inputs with cyan focus states
**Navigation**: Transparent background with neon accent
**Modals**: Dark overlay with animated entrance

## Accessibility

**Contrast Ratios**: Minimum 4.5:1 for all text
**Focus States**: Clear cyan outline on all interactive elements
**Alt Text**: Descriptive text for all images
**Keyboard Navigation**: Full keyboard accessibility
**Screen Reader**: Semantic HTML structure
